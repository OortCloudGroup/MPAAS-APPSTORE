# MPAAS-APPSTORE-SDK 更新日志

## v1.0.0（2026-07-10）

首次正式发布。

### 新增功能

**MPAAS-APPSTORE-SDK 应用市场模块**

- 应用展示：推荐页、分类页、搜索、当月最新应用
- 应用详情：详情页展示，支持 APK / H5 / Web 三种终端类型
- 下载管理：内置 DownloadManager，支持断点续传与进度回调
- 安装管理：自动检测本地安装状态，引导用户完成安装
- 版本更新：支持版本检测与增量更新提示
- 评论评分：支持评论列表、评分、发表评论与回复
- 模块管理：支持应用模块列表初始化与动态更新
- 应用管理页：已安装应用查看与管理

**基础能力**

- 单 AAR 封装：所有功能封装在 `omm-lib` 中，一键集成
- 多源依赖：支持 JitPack、Nexus、GitLab、本地 AAR 四种集成方式
- 权限封装：模块已声明所需权限，宿主无需重复配置
- 网络请求：基于 RxJava2 + Retrofit2 的统一请求框架
- 图片加载：基于 Glide 的图片加载能力
- 本地数据库：基于 LitePal + SQLite 的数据持久化

### 已知限制

- AppStore 模块 Manifest 中声明了 `android:sharedUserId="android.uid.system"`（系统签名），独立使用时需移除
- 存储权限和安装权限需宿主 App 在运行时主动申请
